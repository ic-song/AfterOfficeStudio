# After Office Studio 회사소개

## 1. 개요

After Office Studio는 여러 아이디어와 프로젝트를 구상, 기획, 설계하고 실행 가능한 형태로 발전시키기 위한 개인 작업공간입니다.

하나의 특정 앱이나 서비스가 아니라, 여러 프로젝트 후보를 검토하고 기록하며 필요한 경우 실제 프로젝트로 키워가는 초기 스타트업형 스튜디오입니다.

## 2. 설립 목적

After Office Studio의 목적은 생각을 흩어진 메모로 남겨두지 않고, 검토 가능한 문서와 실행 가능한 계획으로 바꾸는 것입니다.

- 아이디어를 프로젝트 후보로 정리합니다.
- 문제, 사용자, 목적, MVP 범위를 명확히 합니다.
- 수익모델은 확정하지 않고 가능성 중심으로 검토합니다.
- 중요한 판단과 변경 이유를 기록합니다.
- 혼자 시작하더라도 나중에 다른 사람이 합류할 수 있는 구조를 만듭니다.

## 3. 운영 철학

After Office Studio는 빠르게 많이 만드는 것보다 오래 유지할 수 있는 기준과 기록을 중요하게 봅니다.

기획과 실행을 분리하고, 프로젝트가 커지기 전까지 구조를 과하게 나누지 않습니다. 문서는 사람이 이해하기 쉬워야 하며, AI가 보는 규칙은 실행 기준이 명확해야 합니다.

## 4. 주요 활동 영역

- 아이디어 수집과 정리
- 프로젝트 후보 검토
- 서비스화 가능성 검토
- BM 가능성 검토
- MVP 범위 설정
- 개발 가능성 검토
- 운영 기록과 회고 정리

## 5. Notion / Git / AI Agents 역할

### Notion

Notion은 기획과 기록의 중심입니다.

- 아이디어 기록
- 프로젝트 상태 관리
- 회의와 의사결정 기록
- 기획 문서 초안 관리
- 회고 정리

### Git

Git은 실행 가능한 기준과 산출물의 중심입니다.

- AI 작업 규칙 관리
- agent 역할 문서 관리
- 프로젝트 템플릿 관리
- 프로젝트 문서 관리
- 코드가 필요한 프로젝트의 변경 이력 관리

### AI Agents

AI agents는 사람의 판단을 대체하지 않습니다. 사람이 정한 방향과 기준을 바탕으로 기획, 검토, 정리, 기록을 보조합니다.
