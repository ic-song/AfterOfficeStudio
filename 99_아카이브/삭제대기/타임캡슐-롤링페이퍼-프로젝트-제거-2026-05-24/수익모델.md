# 수익모델

## 기본 방향

타임캡슐 롤링페이퍼의 수익화는 광고보다 프리미엄 기능 결제가 더 자연스럽다.

핵심 가치는 메시지를 특정 날짜까지 잠가두고 보관하는 것이다. 따라서 짧은 기간은 무료로 제공하고, 긴 예약 기간이나 장기 보관은 유료화한다.

## 수익구조 후보

| 구분 | 내용 |
| --- | --- |
| 짧은 기간 무료 | 예: 7일 이내 공개, 기본 테마, 제한된 보관 |
| 장기 예약 유료 | 30일, 100일, 1년, 특정 날짜 예약 공개 |
| 장기 보관 유료 | 공개 후 장기 보관 또는 영구 보관 |
| PDF/이미지 저장 유료 | 공개 후 예쁜 PDF나 이미지로 저장 |
| 프리미엄 테마 유료 | 생일, 퇴사, 졸업, 프로젝트 종료 등 이벤트 테마 |
| 비공개/비밀번호 캡슐 | 민감한 메시지를 보호하는 유료 옵션 |
| 단체/기업용 | 참여자 수 확장, 로고/브랜딩, 단체 페이지 |

## 초기 MVP 수익화

초기 MVP에서는 알림톡을 제외한다.

초기에는 아래 정도만 고려한다.

- 짧은 기간 무료
- 긴 기간 보관/예약 유료
- PDF 저장 유료
- 프리미엄 테마 유료

## 알림 정책

초기 MVP에서는 알림톡, SMS, 카카오 채널 연동을 넣지 않는다.

확장 순서:

```text
1. 캘린더 추가 버튼
2. 이메일 알림
3. 유료 알림톡
4. 단체/기업용 알림 옵션
```

알림톡은 발송 비용, 템플릿 심사, 카카오 채널/비즈니스 인증, 공식 딜러사 연동이 필요할 수 있으므로 기본 기능으로 두지 않는다.

## 수익 가설

타임캡슐의 핵심 가치는 특정 날짜까지 메시지를 잠가두고 보관하는 것이다.

짧은 기간은 무료로 제공해 참여 장벽을 낮추고, 긴 예약 기간이나 장기 보관은 감정적 가치와 서버 보관 비용이 생기므로 유료 전환 가능성이 있다.

공개 후 PDF/이미지 저장, 프리미엄 테마, 비공개 캡슐도 기념품 가치와 보안 필요가 있어 유료 기능으로 실험할 수 있다.

## Close 기준 초안

프로젝트는 완성 자체가 목표가 아니라 수익창출이 목표다.

출시 후 일정 기간 동안 실제 수익을 만들지 못하면 개선, 보류, close 중 하나를 결정한다.

기본 검토 기준 초안:

```text
출시 후 90일 동안 유의미한 수익이 없으면 close 검토
```

이 기간과 수익 기준은 실제 출시 전 다시 확정한다.
