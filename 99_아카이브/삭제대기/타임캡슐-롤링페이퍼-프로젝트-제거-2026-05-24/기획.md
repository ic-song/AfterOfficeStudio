# 기획

## 제품 정의

타임캡슐 롤링페이퍼는 여러 사람이 링크로 메시지를 남기고, 설정한 공개일이 되기 전까지 내용을 볼 수 없게 잠가두는 웹서비스다.

## 핵심 사용자 흐름

```text
1. 타임캡슐 만들기
2. 제목, 공개 날짜, 받는 사람 설정
3. 링크 공유
4. 참여자들이 이름과 메시지 작성
5. 공개 전까지 메시지 잠금
6. 공개일 이후 메시지 열람
7. 필요하면 저장 또는 공유
```

## 초기 MVP 범위

- 타임캡슐 생성
- 공개 날짜 설정
- 공유 링크 생성
- 참여자 메시지 작성
- 공개일 전 잠금 화면
- 공개일 이후 메시지 열람
- 기본 테마
- 캘린더 추가 버튼

## MVP에서 제외

- 알림톡
- 문자 SMS
- 카카오 채널 연동
- 다중 알림
- 고급 예약 알림
- 앱 푸시 알림
- 기업용 관리자 기능
- 고급 테마 편집

## MVP 이후 확장 후보

- 이메일 알림
- 프리미엄 테마
- 비공개/비밀번호 캡슐
- PDF/이미지 저장
- 장기 보관
- 알림톡 유료 옵션
- 단체/기업용 캡슐

## 제품 원칙

- 설치 없이 웹에서 바로 쓸 수 있어야 한다.
- 링크 공유만으로 참여할 수 있어야 한다.
- 무료 사용 진입은 가볍게 유지한다.
- 장기 보관, 장기 예약, 저장 기능은 유료 전환 후보로 둔다.
