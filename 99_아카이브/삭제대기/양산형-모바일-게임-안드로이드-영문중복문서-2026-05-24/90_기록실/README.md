# Idea Records

## Purpose

This folder stores idea-level records and supporting documents for the Android mobile game idea.

Use this folder for materials that should remain close to the idea but are not confirmed decisions, core idea overview, or active task tracking.

## Scope

Use this folder for:

- Planning notes
- Meeting notes and discussion summaries
- Research notes
- Review notes
- Drafts that support later idea or project-promotion decisions

Keep these documents outside this folder:

- Confirmed decisions: `../DECISIONS.md`
- Current idea overview: `../PROJECT.md`
- Active tasks: `../TASKS.md`
- AI working context: `../AI_CONTEXT.md`

## Naming

Default Korean record filename:

```text
{topic}-기록-{YYYY-MM-DD}.ko.md
```

Default English or language-neutral record filename:

```text
{topic}-record-{YYYY-MM-DD}.md
```

Use Korean for human-facing operation records by default.
