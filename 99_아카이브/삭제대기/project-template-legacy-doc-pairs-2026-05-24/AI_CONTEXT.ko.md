# AI_CONTEXT.md 한국어 검토본

이 문서는 `50_프로젝트/project-template/AI_CONTEXT.md`의 사람용 한국어 검토본입니다.

## 프로젝트 배경

이 프로젝트가 시작된 배경을 작성합니다.

## 현재 상태

```text
IDEA
```

확정된 내용과 아직 검토 중인 내용을 구분합니다.

## 중요한 규칙

- 추측으로 결정하지 않습니다.
- 확실하지 않은 내용은 확실하지 않다고 표시합니다.
- 사람 승인 없이 파일 삭제, 구조 변경, 정책 변경을 하지 않습니다.
- BM은 확정하지 않고 가능성 중심으로 검토합니다.
- 중요한 결정은 `DECISIONS.md`에 기록합니다.
