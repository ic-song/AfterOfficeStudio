# Project Template

## 프로젝트명

프로젝트명을 입력합니다.

## 프로젝트 목적

이 프로젝트가 왜 필요한지, 어떤 문제를 다루는지 간단히 설명합니다.

## 현재 상태

```text
IDEA
```

초기 상태는 `IDEA`로 시작하고, 진행 상황에 따라 `TODO`, `READY`, `DOING`, `HOLD`, `LIVE`, `DONE`, `ARCHIVE` 중 하나로 변경합니다.

## 관련 문서

- `PROJECT.md`: 프로젝트 개요
- `TASKS.md`: 작업 목록
- `DECISIONS.md`: 결정사항 기록
- `AI_CONTEXT.md`: AI용 프로젝트 맥락
