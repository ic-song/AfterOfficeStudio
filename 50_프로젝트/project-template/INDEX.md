# Project Template Index

## Purpose

This is the AI routing index for a standard promoted project folder.

Use this template after an idea is promoted from `10_아이디어/` to `50_프로젝트/`.

## Template Files

| File | Purpose | Audience |
| --- | --- | --- |
| `안내.md` | Human-facing project overview | Human |
| `기획.md` | Product concept, MVP scope, and deferred items | Human |
| `수익모델.md` | Revenue model, payment path, operating and close criteria | Human |
| `작업.md` | Task list | Human |
| `결정.md` | Confirmed decision log | Human |

## Routing Rules

- For project summary, open `안내.md`.
- For MVP scope, open `기획.md`.
- For monetization or close criteria, open `수익모델.md`.
- For current work, open `작업.md`.
- For confirmed decisions, open `결정.md`.
- This index is for AI routing and should stay English-only.
