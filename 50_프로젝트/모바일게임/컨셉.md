# 판타지 디펜스 모바일게임 컨셉

## 한 줄 컨셉

마법사가 몰려오는 몬스터 웨이브를 막고, 매판 다른 스킬 선택과 영구 성장으로 다음 스테이지에 도전하는 캐주얼 판타지 디펜스 게임.

## 핵심 방향

우리가 만들 게임은 `땡땡마법사`와 같은 게임 골격을 기준으로 잡는다.

단, 이름, 캐릭터, 세계관, 스킬명, 몬스터, UI 표현, 아트 톤은 우리만의 IP로 바꾼다. 핵심은 검증된 플레이 구조를 참고하되, 외형과 설정은 독립적으로 구성하는 것이다.

복잡한 조작보다 `선택`, `성장`, `재도전`에 집중한다.

목표는 짧게 한 판 플레이할 수 있고, 실패해도 조금씩 강해져 다시 도전하고 싶어지는 모바일게임이다.

## 기본 제작 전략

| 구분 | 방향 |
| --- | --- |
| 게임 골격 | 땡땡마법사와 같은 스테이지형 로그라이크 디펜스 구조 |
| 변경 대상 | IP, 세계관, 캐릭터, 스킬명, 몬스터, 배경, UI 톤 |
| 유지할 감각 | 짧은 판, 자동 전투, 스킬 3택, 웨이브 방어, 영구 성장, 재도전 |
| 피해야 할 것 | 원작의 이름, 캐릭터, 이미지, UI, 스킬 표현, 고유 연출을 그대로 복제 |
| 목표 | 익숙한 플레이 골격 위에 다른 판타지 IP를 입혀 빠르게 MVP 검증 |

## 분위기와 세계관

| 요소 | 방향 |
| --- | --- |
| 장르 분위기 | 캐주얼 판타지 디펜스 |
| 세계관 | 새로운 판타지 IP의 주인공이 몬스터 웨이브를 막는 방어전 |
| 톤 | 신비롭고 약간 어두운 던전 감성, 하지만 너무 무겁지 않은 모바일 톤 |
| 주인공 | 마법사, 룬술사, 정령사, 사서, 연금술사 같은 주문 사용자 |
| 적 | 슬라임, 고블린, 언데드, 박쥐, 그림자 몬스터 |
| 배경 | 숲, 폐허, 던전, 마법 탑, 지하 성소 |
| 이펙트 | 화염, 번개, 얼음, 독, 암흑, 폭발 |

## IP 변경 후보

| 후보 | 설명 | 장점 |
| --- | --- | --- |
| 룬 도서관 방어전 | 마법 도서관을 침입하는 몬스터로부터 지키는 설정 | 스킬을 책, 룬, 주문으로 풀기 좋음 |
| 정령 숲 수호자 | 숲의 정령사가 오염된 몬스터를 막는 설정 | 밝은 판타지와 어두운 오염 콘셉트를 함께 사용 가능 |
| 마법 탑 견습생 | 견습 마법사가 마법 탑을 지키며 성장하는 설정 | 성장 구조와 스테이지 진행이 자연스러움 |
| 지하 성소의 룬술사 | 고대 성소에서 깨어난 몬스터를 막는 설정 | UnderDark 같은 어두운 분위기를 살리기 좋음 |

## 플레이 감각

- 플레이어는 직접 복잡하게 조작하지 않는다.
- 기본 공격과 스킬은 자동으로 발동된다.
- 플레이어는 중간중간 스킬, 버프, 보상을 선택한다.
- 선택한 스킬 조합에 따라 매판 전투 감각이 달라진다.
- 실패해도 재화를 얻고 강해져서 다시 도전한다.

## 핵심 루프

```text
스테이지 입장
-> 몬스터 웨이브 시작
-> 자동 공격과 스킬 발동
-> 레벨업 또는 웨이브 보상으로 스킬 선택
-> 보스 또는 최종 웨이브 도전
-> 클리어 또는 실패
-> 골드/재화 획득
-> 영구 강화
-> 다음 스테이지 재도전
```

## MVP 기준

처음에는 아래 범위만으로 게임성을 검증한다.

| 항목 | MVP 방향 |
| --- | --- |
| 스테이지 | 10개 |
| 전투 | 자동 공격 + 적 웨이브 |
| 선택 | 스킬 3택 선택 |
| 스킬 | 6~9개 |
| 성장 | 공격력, 생존력, 골드 획득량 3종 강화 |
| 결과 | 클리어, 실패, 재도전 |
| 수익화 | 보상형 광고 자리만 설계 |

## 수익화 방향

수익화는 가능성으로만 본다. 실제 수익성은 출시 후 지표로 검증해야 한다.

| 지점 | 방식 |
| --- | --- |
| 실패 후 | 광고 보고 부활 또는 추가 보상 |
| 클리어 후 | 광고 보고 보상 2배 |
| 성장 재화 부족 | 소액 재화 패키지 |
| 광고 피로 | 광고 제거 결제 |
| 장기 운영 | 시즌 패스 또는 한정 이벤트 |
