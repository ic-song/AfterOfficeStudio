# Rune Grove Prototype

Timber Rush-style mobile MVP prototype.

## Run

Open `index.html` directly in a browser, or serve this folder with a local static server.

```powershell
python -m http.server 5173
```

Then open:

```text
http://localhost:5173
```

## Controls

- Desktop: WASD or arrow keys.
- Mobile: drag anywhere on the play field.
- Combat is automatic.

## Current Scope

- 60-second run
- Auto attack
- Monster spawning
- XP gems and runes
- Level-up choices
- Permanent upgrades through localStorage
