# 판타지 디펜스 모바일게임 기획 방향

이 문서는 우리가 만들 모바일게임의 기본 방향을 정리한 기획 메모입니다.

`땡땡마법사`, `UnderDark: Defense`처럼 판타지 세계관과 웨이브 디펜스 구조를 참고하되, 최종 목표는 자체 게임 구조를 만드는 것입니다.

## 참고 장르

```text
스테이지형 로그라이크 디펜스
```

## 한 줄 컨셉

플레이어가 마법이나 스킬을 선택하며 몰려오는 적을 막고, 실패하면 성장 재화로 강화해서 다음 스테이지에 도전하는 모바일 디펜스 게임.

## 분위기와 컨셉

마법, 어둠, 몬스터, 던전, 주문을 중심으로 한 판타지 세계관을 기본 분위기로 잡는다.

너무 무겁고 사실적인 다크 판타지보다는, 모바일에서 가볍게 접근할 수 있는 캐주얼 판타지 톤이 적합하다.

| 요소 | 방향 |
| --- | --- |
| 세계관 | 마법사가 몬스터 웨이브를 막는 판타지 방어전 |
| 분위기 | 신비롭고 어두운 던전 감성 + 캐주얼한 모바일 톤 |
| 캐릭터 | 마법사, 정령, 소환수, 룬, 마법 생물 |
| 적 | 슬라임, 고블린, 언데드, 박쥐, 그림자 몬스터 |
| 배경 | 숲, 폐허, 던전, 마법 탑, 지하 성소 |
| 시각 효과 | 화염, 번개, 얼음, 독, 암흑, 폭발 |

## 전체 루프

```text
스테이지 입장
-> 적 웨이브 시작
-> 레벨업 또는 웨이브 보상으로 스킬 선택
-> 스킬 조합으로 몬스터 방어
-> 보스 또는 최종 웨이브 클리어
-> 재화, 장비, 보석 획득
-> 영구 성장
-> 다음 스테이지 도전
```

## 플레이 안 루프

```text
적 등장
-> 자동 공격 또는 스킬 발동
-> 경험치 또는 웨이브 보상 획득
-> 스킬 3개 중 1개 선택
-> 조합 강화
-> 더 강한 웨이브 방어
```

## 플레이 밖 루프

```text
골드 획득
-> 장비 강화
-> 보석 장착
-> 스킬 해금
-> 능력치 상승
-> 막힌 스테이지 재도전
```

## 핵심 재미

- 매판 스킬 선택이 달라진다.
- 좋은 스킬 조합을 찾는 재미가 있다.
- 막힌 구간을 성장으로 뚫는 재미가 있다.
- 숫자가 강해지는 성장감이 있다.
- 짧게 한 판 돌리는 반복성이 있다.

## 핵심 시스템

| 시스템 | 역할 |
| --- | --- |
| 스테이지 | 진행 목표 |
| 웨이브 | 긴장감과 난이도 상승 |
| 스킬 선택 | 매판 변주 |
| 스킬 조합 | 공략과 전략 요소 |
| 장비 강화 | 장기 성장 |
| 보석/룬 | 세팅 다양화 |
| 보스 | 클리어 성취 |
| 일일 보상 | 재방문 유도 |
| 쿠폰/이벤트 | 커뮤니티와 복귀 유도 |

## 수익화 지점

| 지점 | 방식 |
| --- | --- |
| 스테이지 실패 후 | 보상형 광고로 부활 또는 재도전 |
| 클리어 후 | 광고 보고 보상 2배 |
| 일일 보상 | 광고 보고 추가 보상 |
| 성장 재화 부족 | 인앱결제 패키지 |
| 장비/보석 뽑기 | 뽑기 또는 패키지 |
| 광고 제거 | 소액 결제 |
| 시즌/이벤트 | 한정 패스 |

## MVP로 줄인 구조

처음부터 장비, 보석, 쿠폰, 이벤트까지 만들면 범위가 커진다. MVP는 아래 정도로 줄인다.

```text
1. 스테이지 10개
2. 적 웨이브
3. 자동 공격
4. 스킬 선택 6~9개
5. 스테이지 클리어/실패
6. 골드 획득
7. 기본 강화 3종
8. 보상형 광고 자리 설계
```

## MVP 스킬 예시

| 스킬 | 역할 |
| --- | --- |
| 파이어볼 | 단일 공격 |
| 번개 | 랜덤 적 공격 |
| 얼음 | 이동속도 감소 |
| 독 | 지속 피해 |
| 폭발 | 범위 피해 |
| 관통탄 | 여러 적 타격 |

## 양산형 게임틀 요약

```text
짧은 판
자동 전투
스킬 선택
웨이브 방어
영구 성장
재도전
광고/과금 보상
```

- 마법사 디펜스
- 궁수 디펜스
- 로봇 방어전
- 좀비 방어전
- 고양이 왕국 방어전
- 회사원 방어전

## 현재 방향

이 구조는 우리가 만들 모바일 양산형 게임의 초기 방향으로 적합하다.

이 방향을 기준으로 MVP 범위, 수익화 지점, 전투 템포, 아트 톤을 이어서 구체화한다.
