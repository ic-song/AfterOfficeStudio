# 매직서바이벌 게임 분석

조사일: 2026-05-24

참고 링크: [Magic Survival - Google Play](https://play.google.com/store/apps/details?id=com.vkslrzm.Zombie)

## 참고 축

마법 자동전투 생존

## 핵심 구조

사방에서 몰려오는 적을 다양한 마법으로 처치하는 싱글 생존형 게임이다.

플레이어는 직접 공격 버튼을 계속 누르기보다, 자동으로 발동되는 마법과 성장 선택을 통해 오래 버티는 구조를 경험한다.

## 땡땡마법사와 가까운 점

- 마법 스킬 중심의 전투 감각이 있다.
- 자동 공격과 스킬 조합이 핵심 재미가 된다.
- 한 판 안에서 선택한 스킬에 따라 플레이 감각이 달라진다.
- 짧은 반복 플레이와 성장 선택이 모바일에 잘 맞는다.

## 배울 점

- 조작을 줄이고 선택의 재미를 키우면 캐주얼 유저 접근성이 좋아진다.
- 마법 스킬은 시각 효과만 달라도 체감 차이를 만들기 쉽다.
- 같은 전투 구조라도 스킬 조합이 달라지면 반복 피로를 줄일 수 있다.

## 조심할 점

- 방어 거점이 없으면 `디펜스`보다 `생존 액션`에 가까워질 수 있다.
- 화면 안 적과 이펙트가 많아지면 저사양 기기 최적화가 중요해진다.
- 스킬이 너무 많으면 초반 밸런싱과 설명 비용이 커진다.

## MVP 반영 후보

| 항목 | 반영 방식 |
| --- | --- |
| 자동 마법 공격 | 기본 공격은 자동 발동 |
| 3택 스킬 선택 | 레벨업 또는 웨이브 보상 시 스킬 선택 |
| 속성별 마법 | 화염, 번개, 얼음, 독 등으로 역할 분리 |
| 생존 압박 | 웨이브가 진행될수록 적 수와 속도 증가 |

## 우리 게임에 주는 힌트

초기 MVP에서는 `마법사 캐릭터 + 자동 공격 + 3택 스킬 선택`만으로도 충분히 전투 재미를 검증할 수 있다.
